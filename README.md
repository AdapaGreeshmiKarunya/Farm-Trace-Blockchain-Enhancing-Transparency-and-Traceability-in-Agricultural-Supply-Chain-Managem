# Farm Trace Blockchain: Enhancing Transparency and Traceability in Agricultural Supply Chain Management 

## Description
This project uses blockchain technology to enhance transparency and traceability in agricultural supply chains.

## Features
- Smart contract for supply chain tracking
- QR code generation
- Customer verification system

## Technologies
- Python
- Solidity
- Web3

## Files Structure
- contracts/ → Smart contracts
- app.py → Admin interface
- customer_app.py → Customer interface
- deploy.py → Deployment script

## How to Run
1. Deploy contract using deploy.py
2. Run app.py
3. Run customer_app.py

## Note
.env file is not included for security reasons.
